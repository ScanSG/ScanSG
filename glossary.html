Top Singlish Words

Kopi
Coffee. "Kopi-O" = black, no sugar.
Teh-C-Peng
Iced tea with condensed milk.
Shiok
Delicious / amazing feeling.
Bo Jio
Not invited. "You bo jio me!"
Chiong
Rush / go fast. "Chiong this sale!"
Haiyah
Exasperation. "Haiyah, why like that?"
Sibei
Very, extremely. "Sibei kiasu."
Wah Lau
Oh my! Shock. "Wah Lau, so expensive!"

Drink Code Explained

  • Kopi = Coffee
  • Teh = Tea
  • O = Without sugar
  • C = With condensed milk
  • Peng = Iced

Example: Teh-C-Peng = Iced tea with condensed milk.

Lucky. Opposite: siong.
Tahan
To endure. "Can you tahan the heat?"
